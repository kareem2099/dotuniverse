# Detailed Roadmap

This document outlines the planned features and improvements for the dotUniverse portfolio across the 1.x release series.

---

## Phase 1: The Core (v1.0.x) — [CURRENT]

**Focus:** Stability, UX Polish, and "Bug Hunting".

- [x] Context-Aware Tab Completion (Fixed logic for cd, ls, nmap, etc.)
- [x] Smart Command Dictionary (`COMMAND_ARGS` integration)
- [x] Easter Eggs (BSOD, The Final Boss, Matrix, Hack)
- [x] Initial UI/UX & Responsive Design
- [x] Documentation Suite (README, CHANGELOG, ROADMAP)

---

## Phase 2: Terminal 2.0 (v1.1.0) — [Q2 2026]

**Focus:** Making the terminal feel like a real OS.

- [ ] **Piping Support:** `ls | grep` (The ultimate challenge)
- [ ] **SSH Simulation:** Connect to "Remote" servers with hidden content.
- [ ] **VIM Lite:** A secret way to edit virtual files.
- [ ] **Persistent History:** Save commands in `localStorage`.

---

## Phase 3: Visual & Social (v1.2.0 - v1.3.x) — [Q3/Q4 2026]

**Focus:** Aesthetic upgrades and Community.

- [ ] **Live Stats:** Real-time GitHub stars & VS Marketplace downloads.
- [ ] **Dynamic Themes:** Dracula, Nord, and Monokai toggles.
- [ ] **Blog Integration:** Fetching latest dev.to posts via API.

---

## Phase 4: Expansion (v1.4.0+) — [2027+]

**Focus:** Gamification and Internationalization.

- [ ] **Achievement System:** Badges for finding Easter Eggs.
- [ ] **i18n:** Full Arabic (RTL) support for our brothers in the region.
- [ ] **PWA:** Install the dotUniverse terminal as a desktop app.

## Version 1.1.0 — Terminal Enhancements

**Target:** Q2 2026

Enhance the interactive terminal with more commands and features.

### New Terminal Commands

- [ ] `neofetch` — Enhanced with real system info detection
- [ ] `top` / `htop` — Fake process monitor
- [ ] `git` — Simulated git commands (status, log, commit)
- [ ] `ssh` — Fake SSH connection animation
- [ ] `vim` — Simple vim editor simulation
- [ ] `grep` — Search within virtual file system
- [ ] `find` — Find files in virtual file system
- [ ] `wc` — Word count for virtual files
- [ ] `head` / `tail` — Show first/last lines of files
- [ ] `tree` — Directory tree visualization
- [ ] `chmod` — Change file permissions (cosmetic)
- [ ] `ps` — List running processes (fake)

### Terminal Improvements

- [ ] Command piping support (`ls | grep dot`)
- [ ] Output redirection (`cat file > newfile`)
- [ ] Command chaining with `&&` and `;`
- [ ] Persistent command history (localStorage)
- [ ] Multiple terminal tabs
- [ ] Resizable terminal panel
- [ ] Split terminal view
- [ ] Terminal themes (Dracula, Monokai, Nord, etc.)
- [ ] Custom prompt configuration
- [ ] Auto-suggestions based on history
