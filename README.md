## Features

This portfolio website includes:

- **Animated Particle Background** — Canvas-based particle system with connected nodes
- **Glitch Text Effect** — CSS-driven chromatic aberration on the title
- **Interactive Terminal** — Full-featured terminal emulator with 20+ commands
- **Growth Challenge Tracker** — Visual progress bars for April 2026 follower goals
- **VS Code Extension Stats** — Download counts from VS Marketplace & Open VSX
- **AMA Section** — Ask Me Anything with topic categorization
- **Collaboration CTA** — Open for collaboration section
- **Scroll Reveal Animations** — IntersectionObserver-driven fade-up effects
- **Count-up Animations** — Animated statistics on scroll
- **Cursor Glow Trail** — Custom cursor effect with trailing particles

### Terminal Commands

The interactive terminal supports:

```
Contact:  contact, ask, collab
System:   whoami, neofetch, ls, cd, cat, nano, date, history, clear
Fun:      fortune, cowsay, coffee, weather, matrix, hack, ping, calc
Easter:   sudo, apt install, nmap, curl, rm -rf /, exit
```

## Easter Eggs

The terminal contains several hidden features:

- **`rm -rf / --no-preserve-root`** — Triggers a simulated BSOD with card destruction animation, followed by a full recovery
- **`nmap <target>`** — Runs a fake port scan ending with The Final Boss video on port 666
- **`nano <file>`** — Opens a full-screen nano editor simulation
- **`hack`** — Matrix-style random character flood
- **`apt install girlfriend`** — "Package has no installation candidate"
- **`matrix`** — Hue rotation screen effect
- **`exit`** — "There is no escape from FreeRave's terminal"

## Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/kareem2099/dotuniverse.git
cd dotuniverse
```

2. **Open in browser:**

```bash
open index.html
# or
python3 -m http.server 8000
# then visit http://localhost:8000
```

No build step, no dependencies, no frameworks. Just open and go.
